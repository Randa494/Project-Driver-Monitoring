# Driver Behavior Monitoring System(DMS)

##  Project Overview
The Driver Behavior Monitoring System (DBMS) is an AI-based real-time computer vision system designed to improve road safety by monitoring driver behavior and detecting unsafe actions such as seatbelt violation, mobile phone usage, and drowsiness.

The system uses YOLOv8 for object detection and MediaPipe for facial landmark and pose estimation.

---

## Objectives
- Detect seatbelt usage in real-time
- Detect mobile phone distraction
- Detect driver drowsiness using eye and head movement
- Generate real-time alerts for violations
- Compare YOLOv8 and Faster R-CNN performance
- Provide a Streamlit-based dashboard for monitoring

---

##  Technologies Used
- YOLOv8
- Faster R-CNN
- MediaPipe
- OpenCV
- Streamlit
- Python

---

##  Features
- Real-time video processing
- Drowsiness detection (EAR & MAR)
- Seatbelt and phone detection
- Audio alert system
- Streamlit dashboard interface
- Model benchmarking
